1. Definición de Clausura

La definición de clausura es la siguiente: Es la propiedad que el resultado de aplicar una operación entre dos elementos de un conjunto, también pertenece a ese conjunto y por tanto se puede seguir aplicando la misma operación entre los resultados.

2. Clausura

  • suma de números enteros
  • suma de números racionales
  • la creación de listas

3. La función map

La función map es una función de orden superior que toma como argumentos

  • una función f
  • una lista L
  • y devuelve una nueva lista L' donde cada elemento es el resultado de aplicar f a cada elemento de L.

4. Programación de map

(define (map proc items)
  (if (null? items)
      nil
      (cons (proc (car items))
            (map proc (cdr items)))))

5. Programación de reduce

La función reduce es otra función de orden superior que toma como argumentos

  • una función f que tome dos argumentos (binaria)
  • un valor inicial
  • una lista L
  • devuelve el resultado de aplicar la función acumulativamente a los elementos de la lista, de izquierda a derecha, para reducir la lista a un solo valor

6. Programación de reduce

(define (reduce f init lst)
  (if (null? lst)
      init
      (reduce f (f init (car lst)) (cdr lst))))

7. Aplicaciones de reduce

8. Programación de filter

La función filter es otra función de orden superior que toma como argumentos

  • una función \(f\) que toma dos argumentos
  • una lista
  • devuelve una lista con los elementos \(x\) de la lista que dan verdadero \(f(x)\)

9. Programación de filter

(define (filter f lst)
	(if (null? lst)
	    nil
	  (if (f (car lst))
	    (cons (car lst) (filter f (cdr lst)))
	    (filter f (cdr lst))
	    )
	  )
	)

10. SICP

SICP lo aborda como una serie de resultados que van de una etapa a la siguiente.

Estas se acumulan a través de unos datos que van de una función a otra y como se manejan dentro de cada función

11. SICP

(define (accumulate op initial sequence)
  (if (null? sequence)
      initial
      (op (car sequence)
          (accumulate op 
                      initial 
                      (cdr sequence)))))
